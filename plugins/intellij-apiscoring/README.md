## Installation and usage

1. Clone the repository:

	```
    git clone git@github.com:InditexTech/api-scoring-ide-plugins.git
	```

2. Build the application from `plugins/intellij-apiscoring/code`:

    ```bash
    ./gradlew build
    ```
